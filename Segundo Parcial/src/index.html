Segundo Parcial — Resueltos

Respuestas y elementos implementados según el PDF del examen (segundo parcial).

Parte 1 — Accesibilidad y SEO

  • Etiquetas ARIA: navbar y secciones principales tienen role/aria-labels y enlaces con texto claro.
  • Imágenes: todos los img incluyen alt descriptivo y loading="lazy".
  • Meta: meta description presente y viewport optimizado.
  • Contraste: uso de colores con buen contraste y focus-visible mejorado en CSS.

Parte 2 — Diseño Responsive

Implementé grid responsive para menú y galería:

  • Móvil: 1 columna (col-12)
  • Tablet: 2 columnas (col-md-6)
  • Desktop: 3-4 columnas (col-lg-4 / col-lg-3)

Parte 3 — Nueva tarjeta añadida

Se añadió la tarjeta Atol de Elote al grid de menú (imagen, badge, nombre, descripción, precio y botón de ordenar).

Parte 4 — Interactividad (JS)

El archivo main.js incluye:

  • Smooth scroll y actualización de URL.
  • Navbar compacto al hacer scroll.
  • Intersección para animar tarjetas (IntersectionObserver).
  • Toggle ver/ocultar tarjetas extra en móvil.
  • Botón volver arriba, lazy-loading fallback, y notificaciones.

Parte 5 — Accesibilidad técnica

  • Focus visible mejorado con :focus-visible.
  • Uso de roles y aria-label donde aplica (navbar, botones importantes).
  • Formularios con required y validación JS básica.

Parte 6 — Validación

Chequeos realizados:

  • HTML semántico (secciones, header, footer)
  • Imágenes con alt
  • Formularios validados (regex simple para email)

Parte 7 — Performance

  • Lazy loading en imágenes (loading="lazy").
  • Preload de imágenes importantes en JS.
  • Uso de CSS y JS minificados recomendado para producción (aquí está legible para edición).

Parte 8 — Entrega

Archivos incluidos:

  1. index.html — Página completa con la sección "Segundo Parcial — Resueltos".
  2. style.css — Tu CSS (mejorado e integrado).
  3. main.js — Tu JavaScript con todas las funciones solicitadas.

Instrucciones: sube estos archivos al repositorio que te dieron, prueba en localhost y revisa que el botón "Ver Menú Completo" funcione en móvil y que las notificaciones aparezcan al ordenar.
